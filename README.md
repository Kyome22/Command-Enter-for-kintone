# Command-Enter-for-kintone

kintone のコメント投稿を Command + Enter でできるようにする Chrome 拡張です。

Mac では Command + Enter、Windows では Control + Enter でコメントを投稿できます。

## 導入方法

Chrome ウェブストアにて拡張機能をインストールしてください。
